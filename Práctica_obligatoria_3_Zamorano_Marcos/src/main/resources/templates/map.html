Visor cartográfico

Leaflet + OpenStreetMap. Sin API keys. Incluye puntos de interés y sensores sincronizables desde el microservicio Flask.

Remote sensing con Flask: al pulsar el botón de sincronización, Spring Boot llama por REST a python-api:5000/api/sensors, guarda las lecturas en MySQL y las actualiza en este mapa Leaflet. Puedes comprobarlo también desde Postman con POST /api/sensors/sync y desde el navegador en http://127.0.0.1:5000/api/sensors.
