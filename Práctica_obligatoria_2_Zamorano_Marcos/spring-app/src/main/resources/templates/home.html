Sistema distribuido con Spring Boot, Flask y MySQL

Esta aplicación demuestra cómo se capturan, propagan y traducen excepciones en un sistema distribuido con frontend en Spring Boot, API externa en Flask, acceso a base de datos y llamadas a servicios de terceros.

Spring Boot Thymeleaf Flask MySQL Excepciones distribuidas

Frontend Spring Boot

Página principal, login, dashboard y laboratorio web con traducción de errores al usuario.

API Flask

Laboratorio técnico con endpoints de ficheros, base de datos, API externa y timeout.

Base de datos MySQL

Persistencia de usuarios, sesión y soporte para pruebas de acceso a datos.

Estado esperado del sistema:
  • Frontend accesible desde Spring Boot.
  • API Flask operativa y accesible por HTTP.
  • Gestión de login, sesión y token UUID.
  • Pruebas de errores disponibles desde navegador y Postman.

Proyecto académico de sistemas distribuidos. En la memoria final se documentará la arquitectura, los casos de prueba y el tratamiento de excepciones.
