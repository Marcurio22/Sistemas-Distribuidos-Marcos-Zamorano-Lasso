Sistema distribuido con Spring Boot, Flask y MySQL

Esta aplicación demuestra cómo se capturan, propagan y traducen excepciones en un sistema distribuido con frontend en Spring Boot, API externa en Flask, acceso a base de datos y llamadas a servicios de terceros.

Frontend Spring Boot

Página principal, login, registro, dashboard y laboratorio web con traducción de errores.

API Flask

Servicio técnico con endpoints de ficheros, base de datos, Pokémon API y timeout.

Base de datos MySQL

Persistencia de usuarios, token de sesión y soporte para las pruebas de acceso a datos.

Estado esperado del sistema:
  • Frontend accesible desde Spring Boot.
  • API Flask operativa y accesible por HTTP.
  • Gestión de login, registro, sesión y token UUID.
  • Pruebas de errores disponibles desde navegador y Postman.
